# Treinamento de Flutter - Alura


## Comandos úteis

- Verificar se a instalação do flutter está ok:
```
flutter doctor

```

- Criar um projeto:
```
flutter create <NOME_DO_PROJETO>
```

- Rodar o projeto (dentro da pasta do projeto):
```
flutter run
```

- Listar devices:
```
flutter devices
```

- Listar emuladores:
```
flutter emulators
```

- Rodar um emulador:
```
flutter emulators --launch <ID_DO_EMULADOR>
```
